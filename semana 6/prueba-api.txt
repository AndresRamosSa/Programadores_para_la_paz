Al enviar una petición POST a la ruta /registro con el siguiente JSON:

{
“nombre”: “Maria”,
“mensaje”: “Hola comunidad”
}

El servidor responde con:

{
“estado”: “Datos recibidos”,
“nombre”: “Maria”,
“mensaje”: “Hola comunidad”
}

Esto confirma que el servidor recibió correctamente la información enviada por el usuario mediante req.body y la devolvió en formato JSON.
